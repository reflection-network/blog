## Title

Vision & Roadmap

## Description

Nine iterations of infrastructure crystallize into a product vision: a platform for founding and managing AI-native companies.

## Key themes

- From mechanism to metaphor: "AI agents as Git repos" is true but incomplete
- Classical management vocabulary: roles, delegation, oversight, accountability
- Capsule = job description, adapter = hiring process, nix build = onboarding
- Git history = performance record: auditable, rollback-able evolution
- Founder as role: not CEO — skin in the game, bearing responsibility
- New tagline: "Become the founder of your AI-native company"
- Built vs Coming Soon: honest about progress, no aspirational marketing
- Five feature pillars: build team, own everything, run anywhere, evolve, you're the founder

## Image prompt

Abstract futuristic blueprint-style artwork on a deep navy background (#1a1a2e), drawn with thin technical lines in light gray (#e0e0e0).

A long perspective corridor extending into the distance, with construction marks at intervals — waypoints along a planned path. The nearest waypoints are detailed and fully built (completed iterations). Further ones become skeletal, then just dots on the horizon (future work). The corridor has five parallel tracks (the five feature pillars) converging toward a vanishing point. Depth, direction, purpose. The path is clear even where the structures aren't built yet.

Include faint blueprint grid lines and technical construction marks, but keep them minimal and clean.

No text, no labels, no UI elements — purely abstract.

Style: minimal, precise, engineering blueprint aesthetic, high contrast, lots of negative space, strict geometry.

No gradients, no glow effects beyond very subtle line intensity variation.

Mood: calm, systemic, engineered — clarity of direction.
